# Архитектура проекта

## Описание
Бот для Discord, предназначенный для управления сообществом DayZ и интеграции с игровым сервером.

## Технологии
- **Язык**: TypeScript
- **Платформа**: Node.js
- **Библиотека**: Discord.js

## Связи
- **TaskManager**: Внутренняя система управления задачами в Discord.
- **Moderation**: Система управления пользователями (Kick/Ban/Mute).
- **DayZ Sync**: (В разработке) Синхронизация данных с Enforce Script.
